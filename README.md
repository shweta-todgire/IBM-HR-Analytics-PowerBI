# IBM HR Analytics Dashboard | Power BI

## Overview

An interactive HR Analytics Dashboard built using Power BI to analyze employee attrition, workforce demographics, salary trends, career growth, and employee satisfaction using the IBM HR Analytics dataset.

## Key Features

* HR Overview Dashboard
* Attrition Analysis
* Salary & Career Growth Analysis
* Employee Satisfaction Analysis
* Attrition Drivers & Insights
* KPI Cards & Interactive Filters
* Key Influencers & Decomposition Tree

## Tools Used

* Power BI
* DAX
* Power Query
* Data Modeling

## Key Insights

* Attrition Rate: 16.12%
* Overtime significantly impacts employee attrition.
* Sales and R&D departments show the highest attrition.
* Job satisfaction and work-life balance strongly influence retention.
* Business travel frequency is associated with higher attrition.
